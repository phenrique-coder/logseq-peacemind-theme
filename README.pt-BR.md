## Destaques

- **Paleta de Paz Mental**: Projetada para reduzir a fadiga ocular e promover o foco.
- **Modos Claro e Escuro**: Totalmente otimizados para ambos os ambientes.
- **Estética Minimalista**: Linhas limpas e transições suaves para uma escrita sem estresse.
- **Foco em Tipografia**: Hierarquia clara e espaçamento otimizado.
- **Temas Dinâmicos**: Diversas paletas curadas disponíveis instantaneamente.

## Instalação

### Instalação Manual (Modo Desenvolvedor)
Para testar o tema localmente:
1. Abra o Logseq.
2. Vá em `Configurações` > `Avançado` e ative o **Modo Desenvolvedor**.
3. Clique nos três pontos (**...**) no canto superior direito e selecione **Plugins**.
4. Clique no botão **Carregar plugin descompactado**.
5. Selecione a pasta deste repositório.
6. O tema aparecerá na lista e você poderá ativá-lo em `Configurações` > `Aparência`.

## Desenvolvimento
Para customizar as cores, edite o arquivo `palettes.js` para variações globais ou `custom.css` para estilos estruturais.
