- **Typography Focused**: Clear hierarchy and optimized spacing.
- **Dynamic Theming**: Multiple curated palettes available instantly.

## Installation

### Manual Installation (Developer Mode)
To test the theme locally:
1. Open Logseq.
2. Go to `Settings` > `Advanced` and enable **Developer mode**.
3. Click the three dots (**...**) in the top right corner and select **Plugins**.
4. Click the **Load unpacked plugin** button.
5. Select the folder of this repository.
6. The theme will appear in the list and you can activate it in `Settings` > `Appearance`.

## Development
To customize colors, edit the `palettes.js` file for global variations or `custom.css` for structural styles.
