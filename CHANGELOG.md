# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-20

### Added
- Initial release of the **PeaceMind** theme.
- 7 curated color palettes (Sage, Slate, Sand, Rose, Sky, Moss, Clay).
- Dynamic palette selector in Logseq settings.
- Highly optimized Checkbox styling with mode-aware contrast.
- Custom styled task markers (LATER, NOW, DONE).
- Refined typography and spacing for maximum focus.
- Squared button aesthetic for tags and page references.
- Support for both Light and Dark modes.

### Fixed
- Fixed checkbox visibility issues in light mode.
- Fixed dropdown menu jitter and hover effects.
- Improved sidebar truncation and alignment.
- Optimized text colors for eye comfort.
